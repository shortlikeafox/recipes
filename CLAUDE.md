# CLAUDE.md

Instructions for working on this recipe repo.

## Adding a recipe

1. Create a new file in `recipes/` using kebab-case: `recipes/recipe-name-here.md`
2. Use `recipe-template.md` as the starting point
3. Add the recipe to the **Table of Contents** in `README.md` under the appropriate category
4. Update the **Repo Structure** tree in `README.md`

## Recipe file conventions

- **Measurements**: use fractions, not decimals (1/3 cup, not 0.3 cups)
- **Units**: spell out naturally (1 lb, 1 tbsp, 1 tsp) — no "1 pieces egg" style artifacts
- **Ingredient groupings**: split into logical sections (e.g. Patties / Build, Potatoes / Vinaigrette / Mix-ins)
- **Instructions**: clean up any timer clock references (05:00, 60:00) — just write the time naturally
- **Ratings table**: always include at the bottom, even if empty
